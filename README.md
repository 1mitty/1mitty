Case 1: Inland Empire Solar Sales Travel.
Nunez, Ashley.

Project Description:
  This project was programmed to help a solar marketing specialist find out the best travel routes across cities such as Riverside, Moreno Valley, Perris, and Hemet.
  The goal of this program is to find both the shortest path and the most cost-effective route between the cities, ensuring that solar marketing specialist can maximize their time and minimize their travel expenses. 
Program Approaches:
  In this program, Dijkstra's Algorithm is used to calculate the shortest possible route from the starting city, Riverside. I also calculate the possible routes and pick the one with the lowest travel cost and of course, 
  the shortest distance.

How to Use the Program:

  What is needed:
    First you of course will need a C++ compiler to run the program.
    You can then fork/clone the repository.
    You then run the program which will then allow you to see the output.

  Viewing the output:
    The program will first display the adjacency matrix that shows the distances between the cities, it will then display the shortest path and the most low-cost trip.
